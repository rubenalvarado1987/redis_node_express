# Proyecto NodeJS + Redis cache

## Install and Run Redis Server
`brew install redis` 
`redis-server`

## Install And Development server

Run 
 `npm i` 
 `npm start`

API server to `http://localhost:3000/`. 


## DarkSky
 Se debe crear un APIKEY en darksky.net
